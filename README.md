# Book Store Management System

This project is a Spring Boot REST API application developed for managing bookstore inventory.

The application allows users to perform CRUD operations on books such as adding, viewing, updating, and deleting books.

## Technologies Used

- Java
- Spring Boot
- Hibernate (JPA)
- MySQL
- Maven
- Postman
- Git & GitHub

## Features

- Add new books
- View all books
- Search books by name
- Search books by author
- Update book details
- Delete books
- Pagination and sorting support

## Project Architecture

The project follows a layered architecture:

Controller → Service → DAO → Repository → Database

## API Endpoints

POST /book/bookregister – Add new book  
GET /book/getallbooks – Get all books  
GET /book/getbyname – Search book by name  
GET /book/getbyauthor – Search book by author  
PUT /book/updatebook – Update book details  
DELETE /book/deletebook – Delete book
